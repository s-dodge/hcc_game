-----------------------

- locked_exits: set of (room_name, direction) tuples — unlock with .discard()
- item_descriptions: flat dict, keys must match room items list exactly (all lowercase)
- location tracked as local variable in main(), NOT in player dict
- parse() returns (verb, noun) — strips articles, lowercases everything

-------------
Player receives a service ticket. Arrives at HCC IT department to find it empty.
Investigates. Finds keycard (belongs to missing intern) + broken workstation = inciting clue.
IT building is tutorial. Library and Theatre are main game.
Tone: tongue-in-cheek dark academia. Think Control / Night Vale.
